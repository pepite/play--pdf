Welcome on the PDF module sample application!

This application will give you a rough idea of what you can achieve with this Play! module:

#{if !play.mode.isDev()}

Note

You must configure this application to run in ${play.mode.DEV} mode in the application.conf file to be able to access documentation from the sidebar on this page:

application.mode = dev
#{/if}

Examples

Select an action to perform:

  • Print the installation section (~4 pages) of the Play! manual as a PDF document
  • Print the validation section (~10 pages) of the Play! manual as a PDF document
  • Print the whole Play! manual as a PDF document
  • Output the whole Play! manual as a single HTML page (for debugging purpose)
  • Print a PDF containing an embedded PDF file as an image
